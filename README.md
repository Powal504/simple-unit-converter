# Prosty Konwerter Jednostek (React + Vite)

Aplikacja webowa napisana w React z wykorzystaniem Vite, pozwalająca na konwersję jednostek długości, wagi i temperatury. 

## Opis projektu

Ten projekt to prosty, funkcjonalny konwerter jednostek, który umożliwia:

- Wprowadzanie wartości liczbowej do konwersji
- Wybór jednostek wejściowych i wyjściowych
- Wyświetlanie wyniku konwersji wraz z jednostką
- Historię poprzednich konwersji z możliwością sortowania i usuwania wpisów
- Resetowanie formularza

Projekt jest zbudowany w stylu funkcyjnym React, z użyciem funkcji czystych i React Hooks.

## Technologie

- React 18 (funkcyjne komponenty, hooks)
- Vite (szybkie środowisko deweloperskie)
- JavaScript (ES6+)
- CSS (prosty styl)
